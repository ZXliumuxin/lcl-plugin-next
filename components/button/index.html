Button 按钮

常用的操作按钮

基础用法

使用 type 来定义按钮的样式

Default
primary
success
warning
danger
<template>
  <div>
    <l-button>Default</l-button>

    <l-button type="primary">primary</l-button>

    <l-button type="success">success</l-button>

    <l-button type="warning">warning</l-button>

    <l-button type="danger">danger</l-button>
  </div>
</template>
<style>
.l-button + .l-button {
  margin-left: 8px;
}
</style>

禁用状态

使用 disabled 来定义按钮的禁用状态

Default
primary
success
warning
danger
<template>
  <div>
    <l-button disabled>Default</l-button>

    <l-button type="primary" disabled>primary</l-button>

    <l-button type="success" disabled>success</l-button>

    <l-button type="warning" disabled>warning</l-button>

    <l-button type="danger" disabled>danger</l-button>
  </div>
</template>
<style>
.l-button + .l-button {
  margin-left: 8px;
}
</style>

控制大小

使用 size 来定义按钮的大小，默认三种大小：large、medium、small

large Default
medium Default
small Default
large primary
medium primary
small primary
large success
medium success
small success
large warning
medium warning
small warning
large danger
medium danger
small danger
<template>
  <div>
    <l-button size="large">large Default</l-button>
    <l-button size="medium">medium Default</l-button>
    <l-button size="small">small Default</l-button>
  </div>
  <div>
    <l-button size="large" type="primary">large primary</l-button>
    <l-button size="medium" type="primary">medium primary</l-button>
    <l-button size="small" type="primary">small primary</l-button>
  </div>
  <div>
    <l-button size="large" type="success">large success</l-button>
    <l-button size="medium" type="success">medium success</l-button>
    <l-button size="small" type="success">small success</l-button>
  </div>
  <div>
    <l-button size="large" type="warning">large warning</l-button>
    <l-button size="medium" type="warning">medium warning</l-button>
    <l-button size="small" type="warning">small warning</l-button>
  </div>
  <div>
    <l-button size="large" type="danger">large danger</l-button>
    <l-button size="medium" type="danger">medium danger</l-button>
    <l-button size="small" type="danger">small danger</l-button>
  </div>
</template>
<style>
.l-button + .l-button {
  margin-left: 8px;
}
div + div {
  margin-top: 8px;
}
</style>
